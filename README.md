# 🌿 EcoStay Retreat Mobile App

EcoStay Retreat is a mobile application designed for eco-tourism booking and activity management.
Users can explore eco-friendly stays, book rooms, and discover outdoor activities like hikes and tours.

## ✨ Features

* 🔐 User Authentication (Login & Register)
* 🏡 Browse eco-friendly rooms
* 📅 Book rooms with availability check
* 🌄 Explore activities (Hikes, Tours, Bird Watching)
* 🧾 Activity booking system
* 🛠️ Admin panel (Add rooms & activities)
* 🔔 Push notifications
* ☁️ Firebase real-time database

---

## 🏗️ System Architecture

* 📱 Frontend: Android (Java)
* 🔥 Backend: Firebase
* 🧠 Architecture: MVVM

## 🛠️ Technologies Used

* Java (Android)

## 🚀 Getting Started

### Prerequisites

* Android Studio
* Firebase account

### Installation

1. Clone the repository:

```bash
git clone https://github.com/NaveenDilshan1/EcoStay-Retreat-Mobile-App.git
```

2. Open project in Android Studio

3. Connect Firebase:

* Add your `google-services.json`

4. Run the app on emulator or device
