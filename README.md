# 🌿 EcoStay Retreat Mobile Application

## 📱 Overview
EcoStay Retreat is a cross-platform mobile application developed using Flutter and Firebase. It allows users to explore eco-friendly accommodations, book rooms, and reserve outdoor activities like hiking, eco-tours, and bird watching.

---

## 🌱 Features

- 🔐 User Authentication (Login/Register)
- 🏡 Browse eco-friendly rooms
- 📅 Book rooms with availability check
- 🌄 Explore activities (Hikes, Tours, Bird Watching)
- 📌 Activity booking system
- 👨‍💼 Admin panel (Add rooms & activities)
- 🔔 Push notifications
- ☁️ Firebase real-time database

---

## 🛠️ Technologies Used

- Flutter (Dart)
